# MORE POINTERS, ARRAYS AND STRINGS!

### Requirements

* Language: C
* OS: Ubuntu 14.04 LTS
* Compiler: gcc 4.8.4
* Style guidelines: Betty style
* Synopsis

The goal is to learn coding and concepts through the C programming language.

### Concepts to be learned in this project:
* Pointers
* Arrays
* Strings
